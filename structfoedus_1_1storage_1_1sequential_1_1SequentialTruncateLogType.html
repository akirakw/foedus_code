Detailed Description

Log type of TRUNCATE SEQUENTIAL STORAGE operation.

This log corresponds to SequentialStorage::truncate() operation. Like SequentialCreateLogType, this is a metadata operation.

This log type is infrequently triggered, so no optimization. All methods defined in cpp.
