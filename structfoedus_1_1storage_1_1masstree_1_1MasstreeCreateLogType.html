Detailed Description

Log type of CREATE MASSTREE STORAGE operation.

This log corresponds to StorageManager::create_masstree() opereation. CREATE STORAGE has no in-epoch transaction order. It is always processed in a separate epoch from operations for the storage. Thus, we advance epoch right after creating a storage (before following operations).

This log type is infrequently triggered, so no optimization. All methods defined in cpp.
