Detailed Description

Log type of masstree-storage's insert operation.

This is different from hash. In MasstreeBorderPage, we atomically increment the physical record count, set the key, set slot, leaves it as deleted, then unlock the page lock. When we get to here, the record already has key set and has reserved slot.
